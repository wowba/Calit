## :computer: Commit / PR 컨벤션

| 명칭     | 의미                               |
| -------- | ---------------------------------- |
| Feat     | 새로운 기능 추가                   |

| Docs     | 문서 수정                          |
| Design   | CSS 혹은 폰트, 이미지 파일 등 추가 |
| Refactor | 코드 리팩토링                      |
| Chore    | 빌드 업무 수정, 패키지 매니저 수정 |
